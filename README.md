# Multi Role Dashboard - Backend

This is the backend of the Multi Role Dashboard project built with Node.js, Express, and MongoDB.

## 🚀 Features
- User Authentication (Login / Register)
- Role-based access (Admin / User)
- JWT authentication
- REST API

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Cookie Parser
- CORS
